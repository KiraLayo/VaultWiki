# Obsidian

## 概述

基于本地 Markdown 文件的知识管理工具。支持 [[wikilinks]] 双向链接、图谱视图、插件生态。

在 LLM Wiki 体系中，Obsidian 充当 Wiki 的 **IDE**：LLM 是程序员，直接读写 Markdown 文件，人类在 Obsidian 中实时浏览结果。

## 核心特性

- 本地 Markdown 文件，Git 友好
- `[[wikilinks]]` 双向链接
- 图谱视图（可视化页面间连接）
- 插件生态：Dataview、Marp、Web Clipper 等
- YAML frontmatter 支持

## 在 LLM Wiki 中的角色

- Wiki 的显示层和导航层
- 人类通过它浏览 LLM 的编辑结果
- 图谱视图用于发现孤立页面和连接密度
- Web Clipper 用于快速获取源文档
